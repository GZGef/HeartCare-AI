# HeartCare AI - Система предсказания сердечной недостаточности

## 🎯 О проекте

**HeartCare AI** — это профессиональная система машинного обучения для предсказания риска сердечной недостаточности на основе медицинских показателей пациентов. Проект реализован с соблюдением лучших практик программной инженерии и включает в себя полный цикл разработки ML-системы.

## ✨ Ключевые возможности

- 🏥 **Анализ медицинских данных** — обработка и визуализация различных медицинских показателей
- 🤖 **Две ML-модели** — Random Forest и XGBoost для предсказания риска
- 📊 **Визуализация данных** — графики распределения, корреляционные матрицы, матрицы ошибок
- 📈 **Оценка качества** — подробные метрики классификации
- 🧪 **Тестовое прогнозирование** — возможность тестирования на отдельных примерах

## 📊 Результаты

| Модель | Точность на обучении | Точность на тесте |
|--------|---------------------|-------------------|
| Random Forest | 87.42% | 78.00% |
| XGBoost | 77.52% | 75.33% |

**Вывод**: XGBoost показывает более стабильные результаты и рекомендуется для продакшена.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск обучения
```bash
python scripts/train.py
```

### 3. Использование в коде
```python
from src.models.xgboost_model import XGBoostModel

# Загрузка модели
model = XGBoostModel()
model.load_model('results/models/xgboost.joblib')

# Предсказание
prediction = model.predict(new_data)
```
