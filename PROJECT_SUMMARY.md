## 📊 Результаты моделирования

### Точность моделей

| Модель | Точность на обучении | Точность на тесте | Разница |
|--------|---------------------|-------------------|---------|
| Random Forest | 87.42% | 78.00% | -9.42% |
| XGBoost | 77.52% | 75.33% | -2.19% |

### Выводы
1. **XGBoost** показывает более стабильные результаты (разница 2.19%)
2. **Random Forest** демонстрирует переобучение (разница 9.42%)
3. **Рекомендация**: Использовать XGBoost для продакшена

### Важность признаков
Топ-5 наиболее значимых признаков:
1. ST_Slope - наклон ST-сегмента
2. ChestPainType - тип боли в груди
3. Oldpeak - ST-сегмент
4. MaxHR - максимальная ЧСС
5. Age - возраст пациента

## 🏗️ Архитектура проекта

### Структура модулей
```
HeartCare-AI/
├── config/          # Конфигурация (2 файла)
├── src/             # Исходный код (12 файлов)
│   ├── models/      # ML-модели (4 файла)
│   ├── visualization/ # Визуализация (2 файла)
│   └── utils/       # Утилиты (2 файла)
├── scripts/         # Скрипты (1 файл)
├── notebooks/       # Jupyter (1 файл)
├── dataset/         # Данные (1 файл)
├── results/         # Результаты (авто)
└── docs/            # Документация (авто)
```

### Ключевые компоненты

1. **DataLoader** - Загрузка и валидация данных
2. **DataPreprocessor** - Предобработка и подготовка
3. **BaseModel** - Базовый класс для моделей
4. **RandomForestModel** - Модель Random Forest
5. **XGBoostModel** - Модель XGBoost
6. **Visualizer** - Визуализация данных
7. **Helpers** - Вспомогательные функции
