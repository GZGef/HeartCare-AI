## 🚀 Возможности расширения

### Добавление новых моделей
1. Создать `src/models/new_model.py`
2. Наследовать от `BaseModel`
3. Добавить в `config/settings.py`
4. Обновить `scripts/train.py`

### Добавление новых визуализаций
1. Добавить метод в `src/visualization/visualizer.py`
2. Обновить `config/settings.py`
3. Использовать в скриптах

### Добавление новых данных
1. Поместить в `dataset/`
2. Обновить `config/settings.py`
3. Запустить обучение
