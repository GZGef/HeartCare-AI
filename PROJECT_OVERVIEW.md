# Обзор проекта: Система предсказания сердечной недостаточности

## 📋 Краткое описание

Профессиональная система машинного обучения для предсказания риска сердечной недостаточности на основе медицинских показателей пациентов. Проект реализован с соблюдением лучших практик программной инженерии и включает в себя полный цикл разработки ML-системы.

### Принципы проектирования

1. **SOLID Principles**
   - Single Responsibility: Каждый модуль отвечает за одну задачу
   - Open/Closed: Легко расширять, не изменяя существующий код
   - Liskov Substitution: Базовый класс легко заменяется наследниками
   - Interface Segregation: Минимальные интерфейсы
   - Dependency Inversion: Зависимости от абстракций

2. **DRY (Don't Repeat Yourself)**
   - Общая логика вынесена в базовые классы
   - Конфигурация в одном месте
   - Переиспользование кода

3. **KISS (Keep It Simple, Stupid)**
   - Простая и понятная структура
   - Минимум зависимостей
   - Четкая логика

### Точность моделей

|--------|---------------------|-------------------|---------|
| Random Forest | 87.42% | 78.00% | -9.42% |
| XGBoost | 77.52% | 75.33% | -2.19% |

### Анализ результатов

1. **Random Forest**
   - ✅ Высокая точность на обучении (87.42%)
   - ⚠️ Заметное переобучение (разница 9.42%)
   - ✅ Хорошо для базовой модели
   - 📌 Рекомендация: Увеличить регуляризацию

2. **XGBoost**
   - ✅ Стабильные результаты (разница 2.19%)
   - ✅ Лучшая генерализация
   - ✅ Подходит для продакшена
   - 📌 Рекомендация: Основная модель для использования
